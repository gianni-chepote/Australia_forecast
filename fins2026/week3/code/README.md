# Week 3 Week-Local Code

Use this folder for week-local reusable helpers. If the logic becomes useful
across multiple weeks, move it into `fintools/`.

Current Week 3 helpers include:

- `forecast_data.py` for Australia/U.S. input panels
- `forecast_specs.py` for series metadata
- `forecast_pipeline.py` for the benchmark model ladder
- `beginner_forecasting.py` for the Week 3 lecture ladder, including ARX,
  ARMAX, OLS, elastic net, the horse race, and the ensemble
