# Week 3 Prompts

This folder holds reusable Week 3 prompts.

Week 3 has a teaching flow, not just an app build:

1. run the beginner forecasting ladder first
2. keep the focus on the unemployment-change target
3. compare models out of sample
4. only then move to the full app
5. treat the U.S. beginner ladder as a mirror extension, not the default path

- `assistant_starter.md`: default context-loading order plus the Week 3 teaching flow
- add focused prompts here when a repeated lecture, app-audit, or extension task emerges
